HTML TAGS

"h1" creates heading, it is the largest

"h6" creates heading, it is the smallest

"p" creates a new paragraph

    "ul " creates an unordered list
      "ol " creates an ordered list
  • "li " encompasses each list item
"dl " creates a definition list
"div" is used to format block content with CSS
"span" is used to format inline content with CSS
"b" creates bold text (should use instead)

"i" creates italicized text (should use instead)

"code" is used to define source code, usually monospace

"cite" creates a citation, usually processed in italics

"address" creates address section, usually processed in italics

"em" emphasizes a word (usually processed in italics)

clickable text "a " creates a hyperlink to a Uniform Resource Locator

clickable text "a " creates a hyperlink to an email address

"form" defines a form
input type="email" name=name sets the type of textbox

"select" creates a scrolling menu. Size sets the number of menu items visible before user needs to scroll.



input type="checkbox" name=? value=? > Creates a checkbox
Creates a checkbox which is pre-checked.
input type="radio " name=? value=?> creates a radio button
input type="text " name=? size=?> creates a one-line text area. Size sets length, in characters.
input type="submit " value=?> creates a submit button. Value sets the text in the submit button.
input type="reset "> creates a reset button
"table" creates a table
"tr" sets off each row in a table
"td" sets off each cell in a row "th" sets off the table header (a normal cell with bold, centered text)
"br" inserts a line break
"blockquote " puts content in a quote indents text from both sides 
    "pre" creates preformatted text
